    大误：被人嫌弃的感觉是这样的

    smallyu 2017-05-20 大误 

    遇到过自己讨厌的人，也遇到过讨厌自己的人。如果一个人不怎么搭理你，应该不再自找没趣。即使自找没趣了，接下来，不应该在内心里潜意识地抵触它才对么？原本以为我会那样。

    事件1：我发“作业借我抄一下”，没回。（IM）

    事件2：我说“Hi”，立回“你走吧”。（现实）

    事件3：我说“Hi”，没回。（IM）

    事件1和事件3是在IM工具上，事件2是现实世界中。这三件事情，让我好多种感觉交集。不但没有想要远远躲开它，保持距离，反倒感觉事情有点意思。人也是。毕竟没遇到过。


    这些都基于事件之间以及直到目前为止还没有过任何其他联系的基础上。

    时间顺序

    事件1分析。

    是故意还是无意？那个时候完全不清楚什么情况。

    如果是故意的，问题就可怕了。此前完全没有意识到它对我的排斥，所以有可能是一直对我心存不满，瞬间爆发了？（其实这个也挺有意思的，什么事情效果这么明显。坏笑脸。后补）

    如果是瞬间爆发，那我到底做了什么事情，或者身上有哪些让它讨厌的地方，才会造成这样的情况？

    如果不是瞬间爆发，说明它一直都有表现出对我的不满。为什么我没有发觉？是我愚钝了？

    如果是我愚钝了，那我就好好想想愚钝在何处。

    如果不是我愚钝，那就是它对我的不满表现的不够充分和合理。它为什么没能将不满表现好？

    如果不是故意的，又为什么不回？是因为当时没看到，觉得过去一点时间了，没必要再回了？

    事件1分析函数 () 
{
    if (故意的) 
    {
        if (瞬间爆发) 
        {
            对我不满在什么地方？
        } 
        else if (不是瞬间爆发) 
        {
            if (是我愚钝了)
            {
                愚钝在何处？
            }
            else if (不是我愚钝)
            {
                它为什么没能很好表现出不满？
            }
        }
    } 
    else if (不是故意的) 
    {
        if (觉的不需要回了)
        {
            不需要回心态分析函数();
        }
        else
        {
            事件1超越了我的智商。
        }
    }
}

    事件2分析。

    事件2是在现实生活中。我在说出“Hi”的0.5s后，它不假思索的立即说出了“你走吧”。

    ？？？

    即使没遇到它，我停在那里也不是没有理由啊。

    从礼貌的角度讲，不管是否、多么不待见我，都不应该这么说话。不然以后还怎么混社会，走向人生巅峰？

    同时，对我不满在什么地方？

    或者其实本来想要表达的是其他意思。

    事件2分析函数 ()
{
    if (故意的)
    {
        我要教它做人。
    }
    else if (不是故意的)
    {
        不需要回心态分析函数();
    }
}

    事件3分析。

    事件1的分析忽略了事件1的不回是因为抄作业本身的原因。因为事件3没说是抄作业。虽然事件3本来确实就是想抄作业。

    内容和事件1类似。事件3的存在，就是为了从侧面证实事件1，是故意的，还是无意的？

    事件1和事件3相隔时间10天左右。事件3之后，感觉终于放心了。不需要再有疑问，是故意的还是无意的？不回是事实，是故意的。但也有可能进入事件1分析不是故意的流程里的觉得不需要回函数。

    如果是故意的，就是真的厌恶我到打招呼都不想回的程度。那么问题，我到底做了什么我自己没有意识到但却别人看起来惊世骇俗的厌恶至极的事情？

    如果不是故意的，嗯。

    事件3分析函数 ()
{
    if (故意的) 
    {
        对我不满在什么地方？
    }
    else if (不是故意的) 
    {
        if (觉的不需要回了)
        {
            不需要回心态分析函数();
        }
        else
        {
            事件3超越了我的智商。
        }
    }
}

    不需要回心态分析。

    我以前有过不回别人消息的经历，但没有过第一句话就不回的，也没有过连续两次不回的。

    以前没有过被别人不回的经历，可能也就是对这点事情和对那个人感兴趣的原因。

    因为并不熟悉它的性格和行事态度，先从我自己的角度分析一下。

    我不回别人的情况。

    按人，比较熟的人，和相当不熟的人。

    我和它，不生不熟。

    如果是因为觉得熟，没话说。

    如果是因为觉得不熟，好伤。

    如果是因为半生不熟，接前，怎么也应该回的才对，从礼貌的角度，从做人的角度。

    按事，不想做的。

    它不想给我抄作业。

    这是一个很扯的分支。

    不需要回心态分析函数 ()
{
    if (针对人) 
    {
        if (熟)
        {
            不理解。
        }
        else if (半生不熟)
        {
            我要教它做人。
        }
        else if (不熟)
        {
            不理解。
        }
    }
    else if (针对事)
    {
        不需要回心态超越了我的智商。
    }
}

    逻辑顺序

    逻辑顺序本是按照针对人、针对事的角度。突然不想写了。该写的都写了。


    综上，感觉是否有趣并没有取决于它的态度。也许其他感觉也是。

    当然，上面的所有内容全都忽略了最后一种可能：想多了。
